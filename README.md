# Data Analyst SQL Project - Netflix

## Project Overview

This project focuses on analyzing Netflix dataset using SQL to extract meaningful insights.  
It covers 20 real-world business problems and demonstrates strong SQL skills required for data analyst roles.

---

## Dataset Information

The dataset contains information about Netflix content, including:

- Show ID  
- Type (Movie / TV Show)  
- Title  
- Director & Cast  
- Country  
- Date Added  
- Release Year  
- Rating  
- Duration  
- Genre (Listed In)  
- Description  

---

## Objectives

- Perform data exploration and cleaning  
- Solve business-related SQL problems  
- Practice advanced SQL concepts  

---

## SQL Concepts Used

- Aggregations (COUNT, AVG)  
- GROUP BY & ORDER BY  
- Window Functions (RANK, ROW_NUMBER)  
- Subqueries  
- Common Table Expressions (CTEs)  
- String Functions (SPLIT_PART, SUBSTRING, ILIKE)  
- Date Functions  

---

## Key Business Problems Solved

1. Movies vs TV Shows distribution  
2. Most common rating by content type  
3. Content released in a specific year  
4. Top countries with most content  
5. Longest movie  
6. Recent content (last 5 years)  
7. Content by specific director  
8. TV Shows with more than 5 seasons  
9. Content count by genre  
10. Year-wise content analysis (India)  
11. Documentary movies  
12. Missing director data  
13. Actor-based analysis (Salman Khan)  
14. Top actors in Indian movies  
15. Content classification (Good vs Bad)  
16. Keyword-based content analysis  
17. Top 3 longest movies per country  
18. Country-wise movie percentage  
19. Oldest content on Netflix  
20. Advanced subquery comparison  

---

## Outcome

This project helped in strengthening SQL skills and understanding how to solve real-world data problems using structured queries.
